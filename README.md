# Customer Segmentation using Machine Learning

## Project Overview
This project applies K-Means clustering to segment mall customers based on annual income and spending behavior.  
The goal is to help businesses understand customer groups and design targeted marketing strategies.

---

## Dataset
Mall Customers Dataset (Kaggle)

Features:
- Age
- Gender
- Annual Income
- Spending Score

---

## Technologies Used
- Python
- Pandas
- Matplotlib & Seaborn
- Scikit-learn
- K-Means Clustering

---

## Methodology
1. Data Exploration & Visualization
2. Feature Scaling
3. Elbow Method to determine optimal clusters
4. K-Means clustering
5. Business interpretation of customer segments

---

## 📈 Results
Identified 5 customer segments including:
- VIP Customers
- Young Spenders
- High Income Low Spending Customers
- Average Customers
- Budget Customers

---

## Business Value
The analysis provides actionable marketing recommendations to improve customer engagement and revenue.
